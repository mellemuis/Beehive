void testPayloads ( )

function to test payloads of the compleet payload test

Description of unit test.

The unit test is devided over two seperat tests that are part of the compleet payload test. The compleet payload test is a group of tests that test a specific feature of the payload -encoder -decoder class.

Each test is using its own objects for testing. Tests are using the following schematic:

                   +---------+                   +---------+
                   |         |                   |         |
input variable --->| encoder |----> payload ---->| decoder |----> result
                   |         |                   |         |
                   +---------+                   +---------+
